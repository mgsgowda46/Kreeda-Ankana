# 🏏 Kreeda-Ankana – Sports Match Scheduling App

## 📌 Project Overview
Kreeda-Ankana is an Android-based sports management application designed to help local communities organize sports activities efficiently. It provides features for booking sports slots, finding opponents, and scheduling friendly matches for games like cricket and volleyball.

The system solves the problem of unorganized ground usage and lack of coordination between teams in rural and local areas.

---

## 🎯 Problem Statement
In many villages and towns, sports grounds are frequently occupied by the same groups without proper scheduling. Small teams struggle to find opponents and organize matches. Kreeda-Ankana provides a digital solution for fair ground usage and structured match scheduling.

---

## ✨ Features
- 🏟️ Sports ground slot booking  
- 👥 Team creation and management  
- ⚔️ Match request and opponent matching  
- 📅 Automated match scheduling system  
- 🏆 Score tracking and match history  
- 📱 Simple and user-friendly Android UI  

---
